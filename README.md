# ⭐ Star Game  
A fast, lightweight, browser‑based space‑shooter game built using **HTML5**, **CSS**, and **JavaScript**.

## 🚀 Overview  
Star Game is an arcade‑style shooter where the player controls a ship, destroys enemy waves, and survives as long as possible.  
The game is designed to be simple, responsive, and easy to play on both **desktop and mobile browsers**.

## 🎮 Features  
- 🛸 Smooth player movement  
- 👾 Enemy waves & difficulty progression  
- 💥 Shooting mechanics  
- ❤️ Health / lives system  
- 🔊 Sound effects (optional)  
- 📱 Mobile‑friendly UI  
- 🚀 Fast loading — no frameworks required  

## 📂 Project Structure  
```
/assets          → images, sprites, sounds  
/css             → stylesheets  
/js              → game logic files  
index.html       → main game file  
README.md        → documentation  
```

## 🛠️ Technologies  
- **HTML5 Canvas** for rendering  
- **Vanilla JavaScript** (no libraries)  
- **CSS3** for UI & layout  

## 🕹️ How to Play  
- Use **arrow keys** or **touch controls** to move  
- Press **space** (or tap the fire button) to shoot  
- Destroy enemies & survive as long as possible  
- Avoid getting hit or losing all lives  

## 🧑‍💻 Running the Game Locally  
1. Download or clone the repo:  
   ```bash
   git clone https://github.com/tazulritta-stack/STAR-GAME.git
   ```
2. Open `index.html` in any modern browser.

That's it — no installation required.
